# Usage
<!-- usage -->
```sh-session
$ npm install -g gemboot-cli
$ gemboot COMMAND
running command...
$ gemboot (-v|--version|version)
gemboot-cli/1.0.0 linux-x64 node-v10.15.3
$ gemboot --help [COMMAND]
USAGE
  $ gemboot COMMAND
...
```
